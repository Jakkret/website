OpenGaduServer
OpenGaduServer - Jeden z projektów, gdzie jest czasami reserve-engineering, sniffing pakietów i działania klienta. Projekt ten, ma na celu uruchomienie klientów GaduGadu w sposób lokalny. W chwili obecnej, najlepiej jest wspierany klient wersji 5.0.5. Wsparcie dla klienta 6.0 i dalszych nadejdą w późniejszym czasie.

'Infrastruktura serwerowa' jest pisana w języku C, natomiast Python jest używany dla szybkich skryptów (jak np. Lamus - bot GG oparty na API ollama), skrypty te są umiejscowione w folderze src/debug/
